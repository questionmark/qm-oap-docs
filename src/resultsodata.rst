Results OData API
-----------------

..  od:service::    resultsodata
                    Results OData Service

The Results OData API is an OData *service* that provides a RESTful way
of accessing Questionmark results data over the web.

..  note::  The Results OData API is available in Questionmark OnDemand
            and in Perception 5.7 and later.
