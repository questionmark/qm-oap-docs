Results OData Service

The Results OData API is an OData service that provides a RESTful way of accessing Questionmark results data over the web.

Note

The Results OData API is available in Questionmark OnDemand and in Perception 5.7 and later.

These documentation pages are in preparation, in the interim please refer to this guide: https://www.questionmark.com/content/questionmark-results-api-guide
