Delivery OData API
------------------

..  od:service::    deliveryodata
                    Delivery OData Service

The Delivery OData API is an OData *service* that provides a RESTful way
of accessing Questionmark data over the web.

..  note::  The Delivery OData API is currently available only for
            OnDemand users; it will be available for OnPremise in a
            future release.
