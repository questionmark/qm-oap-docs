Authoring OData API
-------------------

..  od:service::    authoringodata
                    Authoring OData Service


..  _OnDemand: https://support.questionmark.com/content/support

..  _OnPremise: https://support.questionmark.com/content/support-onpremise

        .. versionadded::   2020.02

The Authoring OData API is an OData *service* that provides a RESTful
way of accessing Question and Assessment data created in Questionmark
over the web.

..  note::  The Authoring OData API is available for OnDemand_ and
            OnPremise_ users only; it is not available in Questionmark
            Perception
