TABLE A_Outcome_Ex¶

Many of these fields are also accessible through an older view called simple A_Outcome. The difference is the way translations are handled. A_Outcome only provides access to untranslated assessments whereas A_Outcome_Ex introduced the additional fields to indicate the language of the translated outcome.
