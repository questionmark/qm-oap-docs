Authoring OData Service

New in version 2020.02.

The Authoring OData API is an OData service that provides a RESTful way of accessing Question and Assessment data created in Questionmark over the web.

Note

The Authoring OData API is available for OnDemand and OnPremise users only; it is not available in Questionmark Perception
